# Python Scripts API Reference

!!! info "API Documentation"
    This document highlights the primary classes and execution commands for the Python front-end scripts that manage the NMPC simulation in CARLA.

---

## :material-language-python: Simulation Execution

### `nmpc_carla_runner.py`
Executes a single simulation instance. Connects the ego vehicle to the `LocalPlanner` and drives it to the destination.
```bash
# Run a specific exported route scenario
python nmpc_carla_runner.py --scenario scenarios/Town04_Straight.json

# Run with specific spawn points and target speed
python nmpc_carla_runner.py --map Town04 --start 10 --end 45 --speed 80.0
```

### `carla_route_inspector.py`
A Pygame GUI tool to interactively select start and end spawn points. Generates route metadata (distance, junction complexity) and exports it as JSON for the runner.
```bash
python carla_route_inspector.py
# Controls:
# Left Click: Select Waypoint
# Space: Export to JSON
```

## :material-language-python: Planners & Generators

### `LocalPlanner` (`nmpc_core_planner.py`)
The main interface class bridging CARLA state data with the C++ wrapper.
- **`run_step(debug=True)`**: Invoked every simulation tick. Computes current Frenet coordinates, handles early waypoint consumption, dynamically alters target speed (ramping), invokes the NMPC solver, applies gravity compensation for pitch, and returns a `carla.VehicleControl`.

### `ReferenceGenerator` (`reference_generator.py`)
Generates smooth paths and adaptive optimization weights.
- **`generate_reference(ego_state, waypoint_buffer, vx)`**: Computes accumulated curvature ahead. Returns smoothed waypoints (via Laplacian smoothing) and `adaptive_opt` dictionary (containing dynamically modified `Q_D`, `Q_kappa_track`, `R_SteerRate`, etc., to allow flexible lane changes or strict cornering).

## :material-language-python: Validation & Testing

### `nmpc_stress_test.py`
Automatically finds complex intersections and tests the NMPC solver's robustness.
```bash
# Run 50 automated tests focusing on high-curvature routes
python nmpc_stress_test.py --map Town04 --runs 50 --min-angle 60.0 --output results.json
```

### `analyze_stress_results.py`
Analyzes the results generated by the stress test. Categorizes the 'Death Certificate' if the vehicle fails.
```bash
# Generate report for a single session
python analyze_stress_results.py results/session_2026/session_summary.json

# Compare Before & After optimizations
python analyze_stress_results.py results/session_old/session_summary.json results/session_new/session_summary.json
```
